# Express-Banking-System

🏦 Express Banking System
📌 Overview

The Express Banking System is a console-based application developed using JavaScript and Node.js. It allows users to perform basic banking operations such as creating accounts, checking balances, and transferring money. The project demonstrates the use of data structures and algorithms in a real-world scenario.

🎯 Features
Create a new bank account
Check account balance
Transfer money between accounts
Menu-driven user interface
Error handling for invalid operations
🛠️ Technologies Used
JavaScript
Node.js
Readline module
🧠 Data Structures Used
Linked List (Linear Data Structure) – for storing accounts
Binary Search Tree (Non-Linear Data Structure) – for efficient searching
⚙️ Installation & Setup
Clone or download the project
Open terminal in the project folder
Run the following command:
node index.js
▶️ Usage

After running the program, you will see a menu:

1. Create Account
2. Check Balance
3. Transfer Money
4. Exit
Enter your choice
Follow the prompts to perform operations
🧪 Sample Input
Enter choice: 1
Enter account number: 101
Enter balance: 5000
📤 Sample Output
✅ Account created: 101
💰 Balance: 5000
✅ Transfer successful
